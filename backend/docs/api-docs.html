記事一覧を取得

query Parameters
publishStatus
string
Enum: "all" "draft" "published"

公開状態で絞り込み（all: すべて、draft: 下書きのみ、published: 公開済み）

search
any

検索キーワード（title, content を検索）

sortBy
string
Enum: "contentDraft.title:ASC" "contentDraft.title:DESC" "updatedAt:ASC" "updatedAt:DESC"

ソート順

limit
number
Example: limit=20

取得件数（デフォルト: 20、最大: 100）

page
number
Example: page=1

ページ番号（デフォルト: 1）

filter.user.id
number
Example: filter.user.id=1

ユーザーIDで絞り込み

filter.user.organization.id
required
number
Example: filter.user.organization.id=1

組織ID（必須）
